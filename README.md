Netflix Clone

A modern and responsive **Netflix Clone** built using **HTML, CSS, and JavaScript** with dynamic movie sections, search functionality, interactive UI, and responsive design.

Live Demo:  
https://somesh-11.github.io/netflix-clone/


Features

- Netflix-style UI
- Dynamic movie search
- Fully responsive design
- Interactive movie cards
- Dynamic banner updates
- Modern dark theme
- Smooth hover animations
- Organized code structure
- Hosted on AWS EC2
- CI/CD pipeline using GitHub

Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Git & GitHub
- AWS EC2
- GitHub Actions (CI/CD)


Project Structure

```bash
netflix-clone/

├── index.html

```

AWS EC2 Hosting

This project was also hosted on an AWS EC2 instance for cloud deployment practice and server hosting experience.

EC2 Highlights

- Configured Ubuntu EC2 instance
- Installed and configured Nginx
- Deployed static frontend files
- Managed security groups and SSH access
- Served application over public IP


CI/CD Pipeline

A CI/CD pipeline was implemented using GitHub integration for automated deployment workflows.

CI/CD Workflow Includes

- Automatic build/update on push
- GitHub repository integration
- Continuous deployment setup
- Version control with Git
