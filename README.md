 Netflix Clone

A modern and responsive Netflix Clone built using **HTML, CSS, and JavaScript** with dynamic movie sections, interactive UI, search functionality, and smooth user experience.

Live Deployment

This project is successfully deployed on:

- AWS EC2 Instance
- GitHub CI/CD Pipeline

Features

- Responsive Netflix-inspired UI
- Dynamic movie rows
- Hero banner section
- Search functionality
- Interactive movie cards
- Smooth hover animations
- Navbar scroll effect
- Mobile responsive layout
- Dynamic DOM rendering using JavaScript

Tech Stack

Frontend
- HTML5
- CSS3
- JavaScript (Vanilla JS)

Deployment & DevOps
- AWS EC2
- GitHub Actions
- Nginx
- Linux (Ubuntu)
